# Test Archive

This directory contains archived test files used during development and feature validation.

## Contents

### Test Files
- `test_chart_display.py` - Chart display functionality tests
- `test_chart_features.py` - Comprehensive chart visualization feature tests
- `test_gui.py` - GUI component tests
- `test_gui_display.py` - GUI display tests
- `test_integration.py` - Integration tests with actual replay data
- `test_all_action_per_turn.py` - Per-turn action count validation
- `test_buy_food.py` - Buy Food action tests
- `test_buy_food_integer_axis.py` - Buy Food integer axis tests
- `test_buy_food_replay.py` - Buy Food replay integration tests
- `test_buy_food_whole_numbers.py` - Buy Food whole number validation
- `test_buy_pet_final.py` - Buy Pet final validation tests
- `test_buy_pet_whole_numbers.py` - Buy Pet whole number validation
- `test_per_turn_action_count.py` - Per-turn action counting tests
- `test_per_turn_demo.py` - Per-turn demonstration tests
- `test_plot_y_axis.py` - Y-axis plotting tests
- `test_replay_selection.py` - Replay selection tests
- `test_turn_time.py` - Turn time calculation tests
- `test_turn_time_replay.py` - Turn time replay integration tests

## Usage

To run tests from this archive:
```powershell
cd tests_archive
python test_chart_features.py
```

Or from the root directory:
```powershell
python tests_archive/test_chart_features.py
```
